# 参考资料目录

## 用途
此目录存放你的视频脚本项目的参考资料。你可以整理：
- 调研内容和竞品分析
- 个人风格指南和写作模板
- 自定义工作流模板
- 临时文件和草稿想法
- 任何能启发和支持视频创作的资料

## 推荐结构

```
参考资料/
├── 视频/                           # 参考视频
│   ├── 视频分析-1.md
│   └── 竞品拆解.md
├── 账号/                           # 竞品账号
│   ├── 频道分析-1.md
│   └── 头部创作者.md
├── 调研/                           # 通用调研
│   ├── 市场报告.md
│   └── 行业趋势.md
├── 模板/                           # 个人模板（可选）
│   ├── 我的钩子模板.md
│   ├── 我的开场风格.md
│   └── 自定义结构.md
├── 风格/                           # 写作风格指南（可选）
│   ├── 品牌语调.md
│   └── 表达规范.md
└── 临时/                           # 临时文件（可选）
    ├── 草稿想法.md
    └── 快速笔记.md
```

## 使用建议

### 核心目录（自动创建）
1. **视频**: 存放参考视频的分析 - 什么有效、什么无效、为什么
2. **账号**: 追踪竞品频道 - 内容策略、受众、表现数据
3. **调研**: 通用调研资料、市场报告、行业洞察

### 可选目录（按需创建）
4. **模板**: 你的个人脚本模板、钩子公式、开场套路
   - 保存你验证过有效的框架以便复用
   - 自定义适合你领域的结构
5. **风格**: 个人写作风格指南
   - 定义你的品牌语调和表达方式
   - 设置准则保持视频间的一致性
6. **临时**: 临时文件、快速笔记、草稿想法
   - 头脑风暴和未结构化的思考
   - 暂时不确定如何归类的文件

## 如何添加参考资料
- 手动添加文件到对应子目录
- 根据需要创建额外的子目录
- 参考资料可在阶段3（内容调研）时被读取使用
- 使用 Markdown 格式便于阅读和编辑
- 当你提到这些文件时，AI 助手可以读取它们
